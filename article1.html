git的使用

---2018/12/2
git --version 查看git版本
安装好git之后进入github注册账号;
免费的别人可以看你的代码，要进入邮箱激活
然后创建一个仓库
写一个README，别人访问你的仓库，下面会显示的内容
要用markdown语言写
Gitpages给我们提供了免费的主页
新建一个仓库，
命名为:用户名.github.io然后就可以访问了
添加一个README就可以看到了
添加一个index.html
配置git，配置用户名和邮箱
git config --global user.name 用户名
git config --global user.name 邮箱
克隆文件：复制地址 右键点击一个文件夹打开git
git clone 地址
上传一个文件
cd 仓库名
git add 文件名
git commit
跳转到最后：shift+g
下面新加一行：o
退出编辑模式：esc
保存退出：shift+z+z
现在已经提交到.git里面了
推送到远程：git push
把远程的更新下载到本地
git pull
